# Security Policy

## Supported versions

Security fixes are applied to the latest published version on npm.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report it privately via GitHub's [Security Advisories](https://github.com/wavedotso/wave-ui/security/advisories/new) for this repository or by contacting the maintainers to `support@ui.wave.so`.

Include a description, reproduction steps, and the affected version. You'll get an acknowledgement as soon as possible, and we'll coordinate a fix and disclosure with you.

Thank you for helping keep Wave UI and its users safe.
