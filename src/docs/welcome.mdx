## Getting started

Install the package and its peer dependencies:

```bash
npm install @waveso/ui @base-ui/react class-variance-authority clsx tailwind-merge
```

Import and use components:

```tsx
import { Button } from '@waveso/ui/button';

function App() {
  return <Button variant="solid">Get started</Button>;
}
```

## Architecture

- **Base UI** primitives for accessible, unstyled behavior
- **Tailwind CSS v4** for styling via utility classes and CSS variables
- **CVA** for type-safe component variants
- **Motion** for declarative enter/exit animations

## Design tokens

All colors, radii, and spacing are driven by CSS custom properties. Override them in your
`globals.css` to match your brand. See the **Foundation / Colors** page for the full palette.
